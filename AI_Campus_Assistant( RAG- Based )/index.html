Introduction

AI Based

The AI Campus Assistant (RAG-Based) is an intelligent system designed to help students quickly access academic information from official documents. Traditional chatbots depend on predefined answers and cannot understand updated PDF documents. This project uses Retrieval-Augmented Generation (RAG) to retrieve relevant information from university documents like fee structures, syllabus, admission rules and exam guidelines. The system then generates accurate responses using Large Language Models grounded in retrieved content.
